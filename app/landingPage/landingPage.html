GRASPJS

Drag. Drop. Learn

An interactive, real-time drag and drop learning tool for JavaScript beginners.

GraspJS is a draggable sandbox of the very basics of the JavaScript language. Perfect for beginners, Grasp allows users to evaluate common JavaScript code without actually typing anything!



DRAG

All the basics that make the JavaScript language, in draggable block form.

DROP

Drop blocks anywhere in our fully featured canvas and see how code works, without writing a line of code.

LEARN

Sign up and join as a student, and you'll be matched with a teacher in real time who can answer any of your questions.
